Automação de Testes de API - Módulo 33 (EBAC)
Este projeto contém a automação de testes da API ServeRest utilizando Cypress e JavaScript.

🚀 O que foi implementado:
Testes de Contrato: Validação de esquema JSON usando a biblioteca Joi.

Testes Funcionais: Listagem de usuários (GET) e cadastro de novos usuários (POST).

Cenários Negativos: Validação de erro ao tentar cadastrar e-mails duplicados.

Massa de Dados Dinâmica: Uso de lógica JavaScript para gerar e-mails aleatórios e evitar falhas por dados já existentes.

🛠️ Tecnologias
Cypress

Joi

Node.js

🔧 Como rodar os testes
Clone o repositório.

Instale as dependências: npm install.

Abra o Cypress: npx cypress open.
